# Titanic ML Git Demo

Это учебный проект, основанный на уже классическом датасете "Titanic" и решающий определенную бизнес задачу.

## Бизнес-цель
В соревновании на Kaggle предлагается построить модель с наиболее высокой метрикой "доле верных ответов" (accuracy).

Здесь ставиться следующая спасательная задача: при ограниченых ресурсов спасти наибольшее количество выживших при кореблекрушении. 
При этом под "ограниченными ресурсами " подразумевается количество лодок (шлюпок), время. 
Модель спасательной операции упрощена тем фактом, что места и список потенциально выживших известны. 
Список пассажиров имеет характеристики пассажиров, а некоторых из них уже спасены или обнаружены мертвыми (train.csv). 
Остальных же пассажиры (test.csv) должны быть спасены при заданных ограничениях.  

## Задачи
Построить модель, прогнозирующая выживаемость (и степень уверености в прогнозе).
Стоит также понимать, что сформулированная задача относится к задаче ранжирования по оценке модели с дальнейшим отбором.

1 Анализ данных (EDA):
 - изучение структуры датасета
 - анализ распределений признаков
 - исследование дисбаланса классов
 - выявление пропусков и аномалий
2 Препроцессинг:
 - обработка пропусков
 - кодирование категориальных признаков
 - масштабирование числовых признаков
 - построение sklearn Pipeline для исключения утечек данных
3 Базовое обучение и сравнение моделей:
 - Logistic Regression
 - KNN
 - Decision Tree
 - Random Forest
 - Gradient Boosting
 - SVC
 - оценка результатов: валидационные тесты, ROC-AUC, PR-AUC, нализ стабильности (std)
4 Тюнинг гиперпараметров:
 - реализован RandomizedSearchCV
 - использован registry-подход для гиперпараметров
 - сохранены: best params, cv results, tuned model
5 Бизнес-оценка моделей:
 - Precision@N
 - Lift@N
 - оценка на holdout-выборке
6 Финальное обучение:
 - финальная модель обучена на полном датасете
 - сохранена production-версия модели
 - реализован inference-скрипт для новых данных
7 Предсказание на новых данных:
 - на test.csv проводится предсказание с использованием обученной модели
 - рассчитывается вероятность выживания
 - формируется итоговый результат
 - отбираются N пассажиров с наиболее вероятно выжившие
